## ADDED Requirements

### Requirement: 用户可以获取歌词的逐行中文翻译
系统 SHALL 为已保存的原文歌词提供逐行中文翻译，每行原文下方显示对应的中文译文，翻译通过 Google Translate API 生成，响应速度快。

#### Scenario: 翻译成功并展示
- **WHEN** 用户在歌词页面触发"获取翻译"操作且原文歌词已保存
- **THEN** 系统返回逐行翻译结果，原文与译文交替展示（原文一行，译文一行），保持原文分行结构

#### Scenario: 翻译结果为空时的处理
- **WHEN** 翻译 API 返回空内容
- **THEN** 系统 SHALL 显示错误提示"翻译结果为空，请重试"，原文歌词保持正常可见，不切换到翻译视图

#### Scenario: 原文不存在时拒绝翻译
- **WHEN** 用户触发翻译但该歌曲尚未保存原文歌词
- **THEN** 系统 SHALL 提示"请先搜索并保存歌词"

#### Scenario: 翻译 API 失败时的处理
- **WHEN** 翻译请求失败（网络错误、API 错误）
- **THEN** 系统 SHALL 显示错误提示，原文歌词保持正常可见，不切换到翻译视图

### Requirement: 翻译结果可以被保存
系统 SHALL 将翻译结果持久化到对应歌曲记录，避免重复调用外部 API。

#### Scenario: 翻译自动保存
- **WHEN** 翻译成功返回且结果非空
- **THEN** 系统自动保存翻译，用户下次访问无需重新翻译
